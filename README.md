# Mattermost Prompts

A collection of reusable prompts and persona definitions for AI-assisted workflows at Mattermost.

## Prompts

| File | Description |
|------|-------------|
| [techsupport-agent.md](techsupport-agent.md) | System prompt for a Senior Technical Support Engineer agent. Defines tone, format, and operating rules for responding to IT administrator tickets about deploying, operating, and troubleshooting Mattermost. Includes skills for email drafts, KB articles, and feature requests. |
## Usage

These prompts are designed to be used as system prompts for LLMs. Copy the contents of a prompt file and use it as the system instruction for your AI assistant or integration.
