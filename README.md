# Mattermost Prompts

A collection of reusable prompts and persona definitions for AI-assisted workflows at Mattermost.

## Prompts

| File | Description |
|------|-------------|
| [support-reply-persona.md](support-reply-persona.md) | System prompt for a Technical Support Engineer persona. Defines tone, format, and operating rules for responding to IT administrator tickets about deploying, operating, and troubleshooting Mattermost. |

## Usage

These prompts are designed to be used as system prompts for LLMs. Copy the contents of a prompt file and use it as the system instruction for your AI assistant or integration.

## Contributing

To add a new prompt:

1. Create a Markdown file at the repo root with a descriptive name (e.g., `my-persona.md`).
2. Update the table above with a short description.
